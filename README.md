This is a personal side project to learn some concepts about computer graphics, game development and get more comfortable using Rust.

### Features
This is a list containing all of the planned features and their current state.

#### Technical
  - [ ] Terrain
       - [x] Greedy meshing
       - [ ] Procedural Terrain
       - [ ] Saving/Loading
  - [ ] Rendering
       - [ ] Transparency [water, glass]
       - [ ] Custom mesh [foliage, etc.]
       - [ ] Lighting & Shadows
       - [ ] Post processing
  - [ ] Physics
       - [ ] Collision
  - [ ] Filesystem
  - [ ] Graphical User Interface
       - [ ] Text
       - [ ] Container
       - [ ] InputText
       - [ ] Button
       - [ ] Slider
  - [ ] Multiplayer
  - [ ] Modding

#### Gameplay
  - [ ] Player
       - [x] Movement
       - [ ] Hotbar & inventory
       - [ ] Model
       - [ ] Health, hunger & others
  - [ ] Structures
  - [ ] Camera
      - [x] Free Camera
      - [ ] First Person
      - [ ] Third Person
  - [ ] Audio
       - [ ] Sound effect
       - [ ] 3D Sound
       - [ ] Background music
  - [ ] Mobs
      - [ ] Behaviour tree
      - [ ] Path finding


## Controls
  - `WASD`  to move around
  - `Space/Shift` to go up/down
